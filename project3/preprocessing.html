Preprocessing

How do I Implement Preprocessing?

LESS Featurees

Mixins

LESS allows you to "mix in" css properties from existing classes and ids. To "mix in" an existing style to a new selector, type the class name or id name followed by parenthesis.
Examples: .class-name();, #id-name();s

Variables

Variables, like in other programming languages, allow you to store values. In LESS, they can be used to store colors, selector names, urls, and values. Variables do not have to be declared before they are used.

Functions

Functions can be used to perform calculations and change values. There are several prebuilt functions in LESS

Operators

Operators can be used to perform calculations as well as add logic to your stylesheet. There are the standard arithmetic operators as well as comparison or logical operators.

Nested Variables

Nested variables are an excellent way to organize variables within your stylesheet. For example, if you have a specific theme, you can nest all of the variables within that theme into one object. Nested variables can be accessed using dot notation.
